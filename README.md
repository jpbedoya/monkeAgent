# OpenClaw Cloud

SaaS mockup for managed OpenClaw hosting. Built with Next.js 14 (App Router), Tailwind CSS, and Framer Motion. Uses MonkeDAO branding.

## Setup

```bash
npm install
npm run dev
```

Open http://localhost:3000

## Pages

- `/` — Landing page with hero, features, and pricing
- `/signup` — 4-step signup wizard (account, Telegram bot, AI provider, launch)
- `/dashboard` — User dashboard with server status, integrations, activity, and config

## Tech Stack

- Next.js 14 (App Router)
- Tailwind CSS
- Space Grotesk (Google Fonts)
- Framer Motion (provisioning animation)
- All data is mocked — no real backend or API calls
